Module DNAnexus.API


module API: sig .. end
Low-level wrapper functions for each individual method in the DNAnexus API.

These functions are thin wrappers around DNAnexus.api_call for individual API methods. Each function has an optional always_retry argument, which adjusts the retry logic as described in the documentation for DNAnexus.api_call. This argument defaults to true for idempotent methods, and false for others.
See also DNAnexus API Specification
